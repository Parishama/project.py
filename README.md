# EV Chat Widget

A beautiful, modern, and accessible chat widget for Electric Vehicle (EV) help and support. This widget provides an interactive chat interface that can be easily integrated into any website to help users learn about EVs, subsidies, charging, battery care, and more.

## üöÄ Quick Start

### Installation

Simply include the HTML file in your project or copy the widget code directly into your webpage:

```html
<!-- Option 1: Use the standalone file -->
<link rel="stylesheet" href="ev_chat_widget.html">

<!-- Option 2: Copy the widget code directly into your HTML -->
```

### Usage

1. **Standalone**: Open `ev_chat_widget.html` in a web browser
2. **Integration**: Copy the HTML, CSS, and JavaScript sections into your existing webpage
3. **Customization**: Modify the CSS variables in the `:root` selector to match your brand colors

## üé® Customization

The widget uses CSS variables for easy customization:
